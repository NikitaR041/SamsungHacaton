# Сканер персональных данных (ПДн)

Инструмент для автоматического обнаружения персональных данных в файлах различных форматов.
Поддерживает многопоточную обработку и распределённый режим через PySpark.

## Возможности

- Поддерживаемые форматы: `pdf, doc, docx, xls, xlsx, txt, csv, json, html, ipynb, png, jpg, tiff` и другие.
- Детекция:
  - ФИО (русские и латинские)
  - email, телефон
  - паспортные данные, ИНН, СНИЛС
  - банковские карты, расчётные счета
  - биометрические и специальные категории
- Определение уровня защищённости (УЗ-1 … УЗ-4)
- Генерация отчётов в форматах **CSV**, **JSON**, **Markdown**
- Опциональное использование GPU для детекции лиц (YOLOv8)
- Режим распределённой обработки через **PySpark**
